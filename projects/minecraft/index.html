Connecting to my Minecraft Server!

Since standard minecraft is rather bland, I like to run a modded game to make things more fun! If you would like to join in on the fun, follow these directions: (this is a safe and vetted process, so parents have no need to worry!)

  1. Download and install Feed the Beast modpack manager.
  2. Launch Feed The Beast and install the "FTB Vanilla+" modpack
  3. Launch "FTB Vanilla+" modpack
  4. Add the following multiplayer server: {% highlight java %} Server Name=Parker's Minecraft Server Server Address=minecraft.skoviak.com {% endhighlight %}
  5. Hit login to join in on the fun!

If you run into any problems, call Parker's dad and he can try to help!
